# 🏀 NBA Game Predictor

A Python app that predicts NBA game winners using real-time standings data from the NBA API.

## Why I Built This
I built this project to grow my Python skills and combine my interest in basketball with data analytics. As an MIS student, I wanted to work with a real API and see how data can be used to make predictions.

## What It Does
- Fetches live NBA standings directly from NBA.com
- Calculates each team's current win rate
- Applies home-court advantage logic to predict a winner
- Lets you make your own prediction and compare it with the program

## How to Install and Run

**1. Make sure Python is installed:**
python --version

**2. Install the required library:**
pip install nba_api

**3. Run the program:**
python nba_predictor.py

**4. Follow the prompts:**
- Pick a home team and away team by number
- See the prediction based on real win rates
- Enter your own pick and see if you agree!

## Tools Used
- Python 3.12
- nba_api
- NBA.com live standings data
